Se elaboraron dos archivos:
Archivo m_funciones_galton.py
En este archivo se importó la librería Matplotlib para la elaboración del histograma de distribución de las canicas en la simulación.
Se definieron las 2 funciones, la primera establece los parámetros de la simulación del modelo o máquina de Galton de acuerdo con lo requerido, 
12 niveles y 3000 canicas, con importación de “random” para que dar posiciones aleatorias al bajar cada nivel, donde -1 es para moverse a la izquierda
del obstáculo y 1 para la derecha de este.

La segunda función es para el diseño del histograma, con las especificaciones de título y ejes X y Y, tamaño de letras, Font, color y tamaño de éstas.

Archivo LAURA_ROMERO_proyectoM3.PY
En este archivo se encuentra el código del programa llamando o importando las funciones del módulo m_funciones_galton.py para elaborar el histograma 
con las especificaciones establecidas, con el propósito de practicar la elaboración de módulos.
Ambos archivos se encuentran en GitHub.
Nota: en la definición de la función se modifico en un momento dado con parámetro “None” para el número de niveles y cantidad de canicas, y el programa corrió. 
Adicionalmente en el programa se cambiaron los parámetros usando la función con 12 niveles y 3000 canicas y con None. En un primer ejercicio se cambiaron 
en el programa los parámetros con 5 niveles y 300 canicas y en ambos casos funciono y se elaboro la gráfica. 

De igual forma como en las actividades anteriores, se hizo consulta con IA y se siguió la guía de Visual Studio Code.
